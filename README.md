# 操作系统项目CI/CD实用技能套件

本套件包含四个核心技能，用于支持操作系统项目的CI/CD流程：

1. [版本查询技能 (version-query)](./version-query/Skill.md) - 快速查询项目版本信息
2. [版本比较技能 (version-compare)](./version-compare/Skill.md) - 比较不同版本间的差异
3. [构建状态检查技能 (build-status)](./build-status/Skill.md) - 检查CI构建状态
4. [发布协助技能 (release-assist)](./release-assist/Skill.md) - 协助进行版本发布流程

每个技能都有完整的文档，详细描述了功能特性、使用场景和实现要点。
